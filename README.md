# Gestor-de-Opiniones
Este sistema tiene como objetivo crear un sistema de gestión de opiniones similar a las publicaciones de  Facebook, con funcionalidades específicas centradas en la interacción y expresión de opiniones por parte de  los usuarios. 

## Funcionalidades Principales

- Registro y autenticación del usuario
- Gestión de Usuarios
- Sistema de gestión de publicaciones
- Sistema de gestión de comentarios en las publicaciones

##Tecnologías Utilizadas

- bcryptjs
- cors
- dotenv
- express
- express-validator
- helmet
- jsonwebtoken
- mongoose
- morgan
- nodemon
